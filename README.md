# cljs-brainfuck

A Brainfuck interpreter written in ClojureScript on node.js.

## Usage

1.  Get extra node.js modules

        npm install

2.  Compile clojurescript program

        lein cljsbuild once

3.  Run program with examples

        node target/brainfuck.js examples/hello_world.bf
